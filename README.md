# BMW Fair Price Predictor

## Goal
Predict fair market price of BMW vehicles and identify undervalued listings for potential resale profit.

## Tech Stack
- Python
- Pandas / Scikit-learn
- XGBoost
- Jupyter
- VS Code

## Workflow
1. Data preprocessing
2. Feature encoding
3. XGBoost regression
4. Evaluation (MAE / RMSE)
5. Deal detection simulation

## Results
- MAE ≈ €1.5k
- RMSE ≈ €2.6k
